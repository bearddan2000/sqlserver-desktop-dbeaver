# sqlserver-web-adminer

## Description
Creates a small database table
called `dog`.

## Tech stack
- phpAdmin
- ms sql

## Docker stack
- mcr.microsoft.com/mssql/server:2017-latest-ubuntu
- adminer

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
